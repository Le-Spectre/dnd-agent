# D&D 骰子投掷

## 触发条件
- 用户输入 `/roll`
- 用户说 "投骰"、"roll"、"扔个骰子"、"d20" 等

## 功能

### 1. 标准投骰
识别格式：`[数量]d[面数][修饰值][keep/drop]`

- `d20` → 1d20
- `2d6` → 2d6
- `1d20+5` → 1d20+5
- `4d6-2` → 4d6-2
- `1d8` → 1d8
- `4d6kh3` → 4d6 保留最高 3 个（属性值）
- `2d20kh+5` → 优势 +5
- `2d20kl` → 劣势
- `5d6dl1+2` → 5d6 去掉最低 1 个 +2
- `5d6dh1` → 5d6 去掉最高 1 个

### 2. 投骰执行
**必须使用 `roll.sh` 脚本**，传入标准骰子记号：
```bash
./roll.sh <notation>
# 例如：./roll.sh 2d20kh+5
```

### 3. 投骰结果展示
展示格式：
```
🎲 [投骰描述]
结果：[原始骰子数值] = [最终结果]
```

### 4. 常用场景
- 攻击投骰：`/roll 1d20+5` - 攻击
- 伤害投骰：`/roll 2d6+3` - 伤害
- 检定：`/roll 1d20+2` - 感知检定
- 豁免：`/roll 1d20+1` - 敏捷豁免

### 5. 特殊投骰
- 优势骰：投两次取较高 `/roll 2d20kh+5`
- 劣势骰：投两次取较低 `/roll 2d20kl+3`
- 属性值生成：`/roll 4d6kh3`
- 爆发骰：展示每颗骰子结果
