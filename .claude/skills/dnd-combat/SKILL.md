# D&D 战斗系统

## 触发条件
- 用户输入 `/combat`
- 用户说 "进入战斗"、"战斗"、"开始战斗" 等
- 战斗中任何战斗相关输入

## 功能

### 1. 战斗初始化
- **轮次制度**：每轮6秒，行动基于敏捷先攻
- **先攻roll**：所有参战者投1d20+敏捷调整值
- **排序**：从高到低排列，高者先行动
- **先攻条格式**：战斗开始时 AND 每轮开始时，必须输出以下格式的先攻条：

```
Round N
---
1. [先攻值] 角色名称 — 当前HP/最大HP HP, AC 护甲值, 状态
2. [先攻值] 角色名称 — 当前HP/最大HP HP, AC 护甲值, 状态
```

**示例**：
```
Round 1
---
1. [22] 角色A — 17/17 HP, AC 15
2. [21] 角色B — 濒死 4/23 HP, AC 13, 麻痹
3. [15] 地精战士 — 28/28 HP, AC 16
4. [8] 地精弓箭手 — 12/12 HP, AC 13
```

**规则**：
- `N` 从 1 开始，每轮递增
- 严格按先攻条顺序执行行动，不得跳过或重排
- HP 格式：`当前/最大 HP`，0 HP 或濒死时加状态前缀（如 `濒死 0/23 HP`）
- AC 格式：`AC 护甲值`
- 状态标注：受惊, 睡眠, 麻痹, 倒地, 濒死, 束缚, 中毒, 盲目等，无状态时省略

- **回合结构**：
  1. 先攻排序，输出先攻条
  2. 每轮每个生物可执行：动作、bonus action、reaction、移动
  3. 回合结束：重置资源，检查持续效果

### 2. 战斗行动
**动作选项**：
- 攻击（Attack）
- 施法（Cast a Spell）
- 冲刺（Dash）
- 撤离（Disengage）
- 躲避（Dodge）
- 帮助（Help）
- 躲避（Hide）
- 准备（Ready）
- 搜索（Search）
- 使用物品（Use an Object）

**简化动作**：
- 徒手攻击：1d20+力量调整值 vs AC，伤害 1 + 力量调整值
- 盾牌攻击：作为武器

### 3. 攻击流程
1. 攻击者声明攻击目标
2. 投攻击骰：1d20 + 加值 vs 目标AC
3. 若≥AC：命中，投伤害骰
4. 目标投豁免（若有）
5. 应用伤害，降低HP

### 4. 伤害与AC规则
- **护甲AC**：10 + 防具加值 + 敏捷调整值（如适用）
- **命中**：攻击骰 ≥ AC
- **伤害类型**：钝击、穿刺、挥砍（生物）、魔法等

### 5. 生命值与死亡
- **HP降至0**：倒地（stabilize）或死亡
- **负HP**：-1至-最大HP倒地，-最大HP-1死亡
- **死亡豁免**：每回合投1d20，10+成功，1大失败，20苏醒并1HP

### 6. 战斗状态追踪
- 追踪所有参战者当前HP、状态
- 记录已用资源（法术位、能力）
