# D&D 快速查找

## 触发条件
- 用户输入 `/find [关键词]`
- 用户说 "查找火球术"、"查一下这个怪物"、"找战士职业" 等

## 功能

### 1. 查找类型判断
根据关键词自动判断查找类型：
- 法术：包含"术"、"咒"、"magic"等 → `DND5e_chm_md/玩家手册/魔法/`
- 怪物/NPC：描述性词汇 → `DND5e_chm_md/怪物图鉴/`
- 种族：种族名称 → `DND5e_chm_md/玩家手册/种族/`
- 职业：职业名称 → `DND5e_chm_md/玩家手册/职业/`
- 装备/物品：装备名称 → `DND5e_chm_md/玩家手册/装备/`
- 规则/机制：技能名、规则术语 → 相关章节

### 2. 搜索执行
**必须使用 Grep 搜索**，不要直接Read大文件：
```bash
grep -r "关键词" DND5e_chm_md/
```

### 3. 输出格式
查找到内容后，提供：
- **名称**：原文 + 中文
- **类型**：法术/怪物/种族等
- **关键数据**：数值、等级、施法时间等
- **完整描述**：原文摘要
- **DM备注**：该内容的DM使用建议（若适用）

### 4. 示例输出
```
## 火球术 (Fireball)
**等级**：3环 塑能法术
**施法时间**：1动作
**射程**：150英尺
**成分**：V、S、M（一颗硫磺、砷石和硫磺的小球）
**持续时间**：立即

**效果**：
以该地点为中心，半径40英尺的球状区域内爆发出火焰。区域内的每个生物必须进行敏捷豁免。
失败：8d6火焰伤害，成功：半额。

**描述摘要**...
```

## 重要提示
- **禁止直接Read大文件目录** - 必须用Grep定位
- 优先查找中文规则文件
- 搜索无结果时尝试英文关键词

## 输出语言
简体中文（规则内容）
