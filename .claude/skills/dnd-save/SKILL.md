# D&D 战役保存

## 触发条件
- 用户输入 `/save`
- 用户说 "保存游戏"、"保存进度"、"save" 等

## 功能

1. **确认当前战役**
   - 检查是否有正在进行的战役
   - 若无，提示用户先开始或加载战役

2. **更新状态文件**
   - 更新 `campaigns/[战役名]/state.md`：
     - 当前 session 编号
     - 位置
     - 进行中的任务/线索
     - 重要NPC状态
     - 资源变化（gold、道具等）
   - 更新 `campaigns/[战役名]/characters/*.md`：
     - HP变化
     - 等级/经验值
     - 新获得的物品
     - 使用的资源（法术位、能力等）

3. **确认保存**
   - 告知用户保存成功
   - 简要列出本次更新的关键变化

## 重要提示
**每次结束会话时必须保存**。状态文件是唯一的事实来源。

## 输出语言
简体中文

## 参考文件
- `campaigns/[当前战役]/state.md`
- `campaigns/[当前战役]/characters/*.md`
