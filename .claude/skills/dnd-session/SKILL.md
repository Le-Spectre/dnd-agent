# D&D 会话管理

## 触发条件
- 用户输入 `/session`
- 用户说 "开始会话"、"结束会话"、"会话总结" 等

## 功能

### 1. 开始新会话
- 确认当前战役
- 读取上次保存的状态
- 回顾关键事件（可用 `/session recap`）
- 继续游戏或开始新场景

### 2. 结束会话
**必须执行保存流程**：
1. 更新 `state.md`
2. 更新所有 `characters/*.md`
3. 生成会话总结：
   - 本次会话发生的关键事件
   - 经验值/奖励
   - 剧情推进
   - 下次开场提示

### 3. 会话回顾 (`/session recap`)
- 简要列出最近1-3次会话的关键事件
- 帮助玩家回忆当前处境

### 4. 时间追踪
- 世界时间推进
- 短休息/长休息处理
- 资源恢复（生命值、法术位等）

## 输出语言
简体中文

## 参考文件
- `campaigns/[战役名]/state.md` - 当前状态
- `campaigns/[战役名]/characters/*.md` - 角色状态
