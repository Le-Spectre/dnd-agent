# 战斗规则

参考：`DND5e_chm_md/玩家手册/战斗/`、`DND5e_chm_md/玩家手册/状态/`

**所有骰子投掷必须使用 `roll.sh` 脚本。** 详见 CLAUDE.md。

## 先攻与回合顺序

**战斗开始时，你必须按以下步骤进行初始化：**

1. **判定突袭**：若一方未察觉，则处于突袭状态（首回合无法行动，回合结束前不能使用反应）
2. **投先攻**：每个战斗参与者（包括 PC、敌人、GM 控制的 NPC）运行 `roll.sh 1d20+<敏捷调整值>`
3. **排列顺序**：从高到低。平局时比较敏捷调整值，再由玩家决定
4. **设定战斗属性**：确定每个战斗者的 HP 和 AC

**先攻条格式** — 战斗开始时 **且** 每轮开始时，你必须输出以下格式的先攻条：

```
第 N 轮
---
1. [先攻值] 角色名称 — 当前HP/最大HP HP, AC 护甲值, 状态1, 状态2
2. [先攻值] 角色名称 — 当前HP/最大HP HP, AC 护甲值, 状态1, 状态2
3. [先攻值] 角色名称 — 当前HP/最大HP HP, AC 护甲值
```

**先攻条规则**：
- `N` 是轮次编号，从 1 开始
- 按先攻值从高到低排序。战斗中所有角色严格按照此顺序行动，不得跳过或重排
- HP 格式：始终 `当前/最大 HP`（如 `17/23 HP`）。0 HP 或濒死时前缀状态（如 `濒死 0/23 HP`）
- AC 格式：始终 `AC 护甲值`（如 `AC 15`）
- 状态：适用时标注 — 受惊、睡眠、麻痹、倒地、濒死、束缚、中毒、目盲等。无状态时省略
- 每轮开始处理任何行动之前，先更新并输出一次完整的先攻条

示例：

```
第 1 轮
---
1. [22] 角色A — 17/17 HP, AC 15
2. [21] 角色B — 濒死 4/23 HP, AC 13, 麻痹
3. [15] 地精战士 — 28/28 HP, AC 16
4. [8] 地精弓箭手 — 12/12 HP, AC 13
```

## 行动经济

每个回合生物可以进行：

| 类型 | 示例 |
|------|------|
| **移动** | 至多移动速度（可拆分在动作前后） |
| **动作** | 攻击、施法、疾走、撤离、闪避、协助、隐匿、准备、搜索、使用物品 |
| **附赠动作** | 职业特性、部分法术（仅当某能力明确授予时） |
| **反应** | 借机攻击、部分法术（护盾术、法术反制）、准备动作的触发 |
| **自由交互** | 拔出/收起武器、开门、简短说话 |

## 攻击结算

### 近战/远程攻击
1. 玩家声明目标和攻击
2. 运行 `roll.sh 1d20+<攻击加值>` vs 目标 AC
3. **自然 20**：暴击（伤害骰翻倍，如 `roll.sh 2d8+<调整值>`）
4. **自然 1**：自动失手
5. 命中后：运行 `roll.sh <NdM>+<调整值>` 计算伤害

### 法术攻击
- 需要攻击掷的法术：`roll.sh 1d20+<法术调整值>`
- 需要豁免的法术：DC = 8 + 施法属性调整值 + 熟练加值

## 伤害与治疗

- **伤害类型**：钝击、穿刺、挥砍、火焰、冷冻、闪电、雷鸣、毒素、强酸、黯蚀、光耀、力场、心灵
- **抗性**：伤害减半
- **易伤**：伤害翻倍
- **免疫**：不受伤害

### 0 HP 状态
1. 倒地并失去意识
2. 在你的回合开始时进行死亡豁免
3. **死亡豁免**：`roll.sh 1d20`，无修正
   - 10 以上：成功
   - 9 或以下：失败
   - 自然 20：恢复 1 HP，苏醒
   - 自然 1：两次失败
4. **3 次成功**：稳定（昏迷但不濒死）
5. **3 次失败**：死亡
6. 0 HP 时受到伤害 = 一次失败（暴击 = 两次失败）

### 严重伤害
0 HP 后剩余伤害大于等于最大 HP：立即死亡。

## 常见状态

| 状态 | 效果 |
|------|------|
| **目盲** | 依赖视觉的检定自动失败，攻击劣势，对你攻击优势 |
| **魅惑** | 无法攻击魅惑者，魅惑者社交检定有优势 |
| **受惊** | 恐惧源可见时攻击劣势，无法自愿靠近 |
| **擒抱** | 速度为 0，擒抱者失能或你被移出范围时结束 |
| **失能** | 无法进行动作和反应 |
| **隐形** | 重度遮蔽，攻击优势，对你攻击劣势 |
| **麻痹** | 失能，力量/敏捷豁免自动失败，对你攻击优势，5 尺内命中必暴击 |
| **中毒** | 攻击和能力检定劣势 |
| **倒地** | 攻击劣势，近战对你优势，远程对你劣势，起身消耗一半移动力 |
| **束缚** | 速度为 0，攻击劣势，对你攻击优势，敏捷豁免劣势 |
| **震慑** | 失能，力量/敏捷豁免自动失败，对你攻击优势 |
| **昏迷** | 失能，掉落物品，倒地，力量/敏捷豁免自动失败，对你攻击优势，5 尺内命中必暴击 |

## 掩体

- **半掩体**（AC +2，敏捷豁免 +2）：矮墙、家具、生物
- **四分之三掩体**（AC +5，敏捷豁免 +5）：铁闸门、箭孔
- **完全掩体**：无法被直接选为目标

## 借机攻击

触发条件：敌人在未撤离的情况下离开你的触及范围
- 使用反应
- 一次近战攻击

## 环境危害

生动地描述危险并公平预告：
- **困难地形**：移动速度减半
- **坠落**：每 10 尺 1d6 钝击伤害（最多 20d6）
- **窒息**：分钟数 = 1 + 体质调整值，之后体质调整值轮后降至 0 HP
- **火焰**：通常 1d10 火焰伤害，可能点燃可燃物

## 法术施放

### 法术位追踪
每次施法后，**立即**更新法术位：
```
施法前：1环 × 2
施放魔法飞弹后：1环 × 1
施放护盾术后：1环 × 0
```

### 戏法
- 无限使用，不消耗法术位
- 包括攻击戏法（雷鸣斩、火焰箭等）

## 死亡豁免与 0 HP

死亡豁免在 0 HP 时自动于**回合开始**时进行：
- 投 1d20，无修正
- 10+ = 1 次成功
- 9 或以下 = 1 次失败
- 自然 20 = 恢复 1 HP，苏醒
- 自然 1 = 2 次失败

死亡豁免由 DM 自动投骰，除非玩家有理由延迟。

## 探索中的主动检定

始终提示相关检定，不要直接描述结果。

不要说："你听到楼上有脚步声。"
应该说："进行感知检定。"

GM 描述需要哪种感官，玩家决定使用哪个技能。

| 情况 | 检定 |
|------|------|
| 听到什么 | 感知 |
| 注意到细节 | 调查 或 感知 |
| 感知魔法 | 奥术 或 宗教 |
| 辨认符号 | 宗教 或 奥术 |
| 潜行通过 | 敏捷（隐匿） |

被动检定（如被动感知）直接与 DC 比较，无需投骰。

## 战斗叙述

不要只报数字。让战斗有真实感：

**机械**："你命中了，造成 8 点伤害。"

**更好**："你的剑刃找到了兽人肩甲和胸甲之间的缝隙。它咆哮着，黑色血液从伤口涌出。[8 点伤害]"

描述：
- 攻击的物理感
- 敌人的反应（痛苦、恐惧、愤怒）
- 环境后果
- 战局的变化
