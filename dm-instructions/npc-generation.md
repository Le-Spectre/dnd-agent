## 人格生成

### 快速特质（各 d6 骰）

**举止**：
1. 紧张、坐立不安
2. 粗鲁、简短
3. 温暖、热情
4. 多疑、戒备
5. 傲慢、不屑
6. 疲惫、无奈

**动机**：
1. 生存——混日子
2. 贪婪——财富至上
3. 忠诚——效忠于人或信仰
4. 野心——向上爬
5. 复仇——清算旧账
6. 保护——守护某人安全

**秘密**：
1. 隐瞒犯罪过去
2. 欠危险人物的债
3. 知道不该知道的事
4. 不是自称的那样
5. 背叛过所爱之人
6. 身患绝症/诅咒

### 外貌特征（d8）
1. 缺手指/耳朵
2. 明显伤疤
3. 神经质的小动作
4. 异色瞳
5. 跛脚
6. 不停嚼东西
7. 说话时手舞足蹈
8. 从不眼神交流

### 语音/说话模式
- 说话慢，字斟句酌
- 语速快，难以插话
- 什么都用耳语
- 不合时宜地大笑
- 用词华丽
- 用第三人称说话
- 重口音
- 紧张时结巴

## 扮演指南

### 让 NPC 令人难忘

1. **即时印象**：一个立即可见或可听的特征，让人一眼辨认
2. **明确需求**：他们从这次对话/场景中想要什么？
3. **障碍**：什么阻止了他们得到它？
4. **人性**：一个令人同情或有共鸣的细节

### 对话原则

- **不要独白**：NPC 应该回应，而不是说教
- **对玩家做出反应**：根据 PC 的态度调整语气
- **信息有代价**：重要信息需要说服、支付或人情交换
- **知识不完美**：NPC 不是什么都知道；谣言可能是错的

### NPC 关系

追踪 NPC 对队伍的态度：
- **敌对**：会对抗，可能攻击
- **不友好**：不会帮忙，可能阻挠
- **中立**：无所谓
- **友善**：愿意在合理范围内帮忙
- **同盟**：主动支持，愿意为他们冒险

行为会改变关系。做出的承诺应该被追踪。

## 命名 NPC 模板

重要常驻 NPC：

```markdown
## [名称]

**身份**：[职业/职位]
**位置**：[通常在哪里]
**外貌**：[简短外貌描述]

**性格**：[2-3 个关键特质]
**动机**：[驱动力]
**秘密**：[隐藏的真相]
**说话方式**：[如何说话]

**与队伍关系**：[当前立场]
**用途**：[能为 PC 提供什么]

**数据**（如需要）：
- 使用 SRD 中的 [X] 数据块
- 或：AC [X], HP [X], 攻击 +[X] 伤害 [X]
```

## 即兴 NPC

当玩家与意外 NPC 互动时：

1. **立即赋予特征**：选一个举止、一个外貌特征
2. **确定知识水平**：他们 realistically 知道什么？
3. **记下来**：如果互动有意义，记录以备后续一致
4. **给他们名字**：即使是小 NPC 有了名字也会令人难忘

## 势力与组织

创建组织时：

1. **目标**：势力想要什么？
2. **手段**：他们如何追求？
3. **标志**：视觉标识
4. **领袖**：顶层的命名 NPC
5. **成员类型**：PC 会遇到什么样的成员？
6. **资源**：他们能调动什么？
7. **敌人**：谁在反对他们？
