## 专注

### 规则
- 同一时间只能专注一个法术
- 施放另一个专注法术会结束前一个
- 受到伤害时：体质豁免（`roll.sh 1d20+<体质调整值>`，DC = 10 或伤害的一半，取较高值）
- 被失能或杀死时结束专注
- 环境干扰（DM 裁定）

### 追踪
施放专注法术时明确标记：
```
[专注中：祝福术，剩余 8 轮]
```

施法者受到伤害时，提示进行专注豁免。

## 法术成分

| 成分 | 要求 |
|------|------|
| **V**（言语）：必须能说话；被沉默时失效 |
| **S**：（姿势）：至少一只空闲的手；被束缚时失效 |
| **M**：（材料）：成分袋或施法法器或特定物品 |

### 材料成分规则
- 没有金币成本的成分：用法器或成分袋替代
- 有金币成本的成分：必须持有特定物品
- 被消耗的成分：施法后必须替换
- 不被消耗的成分：可无限重复使用

示例：*复生术* 需要价值 300 金币的钻石，且**会被消耗**。

## 仪式施法

### 规则
- 法术必须有仪式标签
- 比正常施法时间多 10 分钟
- 不消耗法术位
- 必须已准备该法术（法师则需在法术书中）

### 拥有仪式施法的职业
- **吟游诗人**：必须已知该法术
- **牧师**：必须已准备该法术
- **德鲁伊**：必须已准备该法术
- **法师**：法术必须在法术书中（不需要准备）

## 施法时间

| 时间 | 效果 |
|------|------|
| **1 动作**：标准战斗施法 |
| **1 附赠动作**：也可进行动作，但该回合动作只能施放戏法 |
| **1 反应**：由特定条件触发 |
| **1 分钟**：战斗中 10 轮，或叙事时间 |
| **10 分钟**：仪式时间或准备 |
| **1 小时+**：延长施法，通常非战斗 |

### 附赠动作施法规则
如果你以附赠动作施放法术，则该回合你的动作只能施放戏法。

## 法术攻击与豁免

### 攻击掷
使用 `roll.sh 1d20+<法术调整值>`（施法属性调整值 + 熟练加值）

### 豁免 DC
```
8 + 施法属性调整值 + 熟练加值
```

## 法术反制与解除魔法

### 法术反制（Counterspell）
- 反应动作，当你看到 60 尺内有生物施法时
- 使用 3 环法术位：自动反制 3 环或更低法术
- 更高等级法术：能力检定 DC = 10 + 法术等级
- 可升环施放法术反制来自动反制该等级或更低的法术

### 解除魔法（Dispel Magic）
- 动作，结束目标身上的法术
- 使用 3 环法术位：结束 3 环或更低法术
- 更高等级法术：能力检定 DC = 10 + 法术等级
- 可升环施放来自动结束该等级或更低的法术

## 描述魔法

让施法有仪式感：

**机械**："我施放火球术。"

**更好**："薇拉在空中快速画出一个符号，力量的话语从她唇间倾泻而出，一颗橙色光珠从她指尖射向地精群……"

考虑：
- **言语**：什么语言？什么语调？古代词汇、吟唱短语、喊出的命令？
- **姿势**：优雅的手势？激烈的动作？微妙的手指移动？
- **视觉**：魔法看起来怎样？颜色、质感、声音？
- **感觉**：有臭氧味吗？感觉冰冷吗？让手臂上的汗毛竖起？

## 法术位恢复

| 方法 | 效果 |
|------|------|
| **长休**：所有法术位恢复 |
| **奥术恢复**（法师）：短休，恢复法术位 = 法师等级一半（向上取整），每天一次 |
| **魔力之源**（术士）：术法点与法术位互相转换 |
| **契约魔法**（邪术师）：所有契约法术位在短休后恢复 |

## 常见法术裁定

### 视线与效果线
- 大多数法术需要到目标的清晰路径
- 玻璃挡住效果线但不挡视线
- 完全掩体两者都挡

### 区域效果
- **锥形**：从起始点出发，逐渐变宽
- **立方体**：起始点在一个面上
- **圆柱体**：起始点在圆形底座中心
- **线形**：从起始点沿选择方向直线延伸
- **球体**：起始点在中心

### 叠加
- 同一法术来自不同来源不叠加（取最有效的）
- 不同法术可以叠加
- 同一法术不同等级：使用更高等级效果
